## Perché il nome "RoundTableItaly"

I progetti qui ospitati nascono come software sviluppato per essere utilizzato da Round Table
Italia (es. il portale eventi TW-Events). Il nome dell'organizzazione riflette questo contesto
d'uso storico. La titolarità del codice è oggi personale e **in transizione** verso RTIT: lo
stato aggiornato è dichiarato nel file `LICENSE` di ciascun repository.

## Stato della cessione

Un atto scritto di cessione dei diritti patrimoniali a Round Table Italia (RTIT) è in
preparazione e non è ancora firmato. Fino alla firma e alla ratifica del Comitato Nazionale,
Riccardo Dal Fiume resta il copyright holder. Nessun accordo di servizio o licenza d'uso
sostituisce o anticipa gli effetti di quella cessione.
