# RoundTableItaly (GitHub organization)

**Ownership note**: this GitHub organization is created, owned, and administered personally by
**Riccardo Dal Fiume**. It is not (yet) an entity or institutional asset of Round Table Italia
(RTIT), even though the name references it in the title.

The assignment of the economic rights over the software hosted here to RTIT is **agreed and in
progress**, pending signature and ratification by the RTIT National Committee. Until that act,
Riccardo Dal Fiume remains the copyright holder, as stated in the `LICENSE` file of each
repository. Do not treat the assignment as already effective. Once signed and ratified, ownership
will pass to RTIT (with Riccardo Dal Fiume retaining inalienable moral rights of authorship and a
broad license-back).

## Why the name "RoundTableItaly"

The projects hosted here originated as software developed for use by Round Table Italia
(e.g. the TW-Events portal). The organization name reflects this historical context of use.
Ownership of the code is currently personal and **in transition** to RTIT: the up-to-date status
is declared in the `LICENSE` file of each repository.

## Status of the assignment

A written assignment of the economic rights to Round Table Italia (RTIT) is being prepared and is
not yet signed. Until signature and ratification by the National Committee, Riccardo Dal Fiume
remains the copyright holder. No service or end-user license agreement replaces or anticipates
the effects of that assignment.

## External contributors

External contributions (code, documentation, material submitted via pull request) are assigned
**directly to RTIT** under the Contributor License Agreement (CLA) of each repository. Before a
pull request can be accepted, electronic signature of the CLA is required: see `CONTRIBUTING.md`
and `CLA.md` in the relevant repository.
